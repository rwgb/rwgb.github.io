## Why This Site?

I wanted a central hub where everything lives together. GitHub is great for code, but I needed a space to:

1. **Write longer-form content** without platform restrictions
2. **Own my content** fully
3. **Have complete creative control** over design and layout
4. **Create a portfolio** that represents me

## The Tech Stack

For those curious, this site is built with:
- **Hugo** - Static site generator for blazing fast performance
- **Congo Theme** - Beautiful, dark-mode-first design
- **GitHub Pages** - Free hosting with automatic deployments
- **GitHub Actions** - CI/CD pipeline for smooth updates

The entire site is [open source on GitHub](https://github.com/rwgb/rwgb.github.io), so feel free to learn from it or fork it!
