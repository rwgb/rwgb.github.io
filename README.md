# Ralph Brynard's Personal Website

Personal website built with Hugo and the Congo theme, hosted on GitHub Pages.

## Site URL

https://rwgb.github.io/

## Local Development

```bash
# Start development server
hugo server -D

# Visit http://localhost:1313
```

## Building

```bash
# Build the site
hugo --minify
```

## Deployment

The site automatically deploys to GitHub Pages via GitHub Actions when pushing to the `main` branch.

## Tech Stack

- **Hugo** - Static site generator
- **Congo Theme** - Modern, responsive theme
- **GitHub Pages** - Hosting
- **GitHub Actions** - CI/CD
